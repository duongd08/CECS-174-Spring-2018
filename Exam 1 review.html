CECS 174 - Exam 1 review

Python basics

  • General execution model of a Python program: top-down, statement by statement
  • Vocabulary: interpreter, statement, expression, operator, operand, function, argument, parameter, condition, comparison, type
  • Declaring, assigning, and using variables
  • Variable types: int, float, str, bool
  • Arithmetic operators: + - * / // %
  • Comparison operators: < > <= >= == !=
  • Compound conditional operators: and or not
  • Input and output, converting input to specific data types

Control flow

  • if, elif, else statements
  • while loops
  • "break" statement

Functions

  • Calling functions and passing arguments to the parameters
  • Difference between arguments and parameters
  • Creating your own functions and returning values from them
  • Be able to write functions like: max3, is_factor, calculate_distance (Lab 2), etc.
  • Answer a question about "why" functions are useful in computing

Random numbers

  • Generate a random integer or float in a given range
  • Write a function or program using a random number generator as instructed

Lists

  • Basics of lists: create a list containing specific values. Append a value to an existing list.
  • Use a for loop to iterate through each element of a list.
  • Write a simple function with a list: find the largest element of a list (by hand, without using Python's max function); find the mean (average) of the numbers in a list (again, by hand); print the elements of a list to the screen, one per line.
